App PWA VR de Newton

Primera Ley: inercia y rozamiento

Versión corregida: no usa librerías externas. Los botones funcionan directamente en GitHub Pages.

Activa el modo lentes, calibra la vista y usa el suavizado para reducir el temblor del celular.

Cómo usar la app VR

1. Abre el enlace desde el celular.

2. Presiona “Entrar al laboratorio”.

3. Presiona “Modo lentes”.

4. Mira al frente y presiona “Calibrar vista”.

5. Coloca el celular horizontalmente dentro del visor VR.

6. Si la imagen tiembla, baja la sensibilidad aumentando el suavizado lentamente.
